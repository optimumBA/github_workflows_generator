# mix github_workflows.generate

## Installation

Add `github_workflows_generator` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:github_workflows_generator, "~> 0.1"}
  ]
end
```

## Usage

```bash
mix github_workflows.generate
```

To see available options, run:

```bash
mix help github_workflows.generate
```
